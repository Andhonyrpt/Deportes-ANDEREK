# Deportes-ANDEREK - Plataforma E-commerce

Este repositorio contiene la solución completa de una plataforma e-commerce, dividida en un backend robusto y un frontend moderno.

## Estructura del Proyecto

- **[ecommerce-api](file:///c:/Users/Usuario/Documents/Deportes-ANDEREK/ecommerce-api)**: Backend construido con Node.js, Express y MongoDB. Incluye una suite completa de pruebas unitarias e integración con Vitest.
- **[ecommerce-app](file:///c:/Users/Usuario/Documents/Deportes-ANDEREK/ecommerce-app)**: Frontend desarrollado en React. Gestión de estado global, carrito persistente y flujo de checkout dinámico.
